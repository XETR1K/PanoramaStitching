# Проект "Panorama Stitching for Multiple Video Streams"

Этот проект представляет собой программу для создания панорамных изображений из нескольких потоков видео с использованием данных с нескольких камер. Основной целью программы является объединение видеопотоков с перекрывающимися областями с различных камер в единый панорамный видеопоток.

## Цель проекта

Цель проекта заключается в разработке алгоритмов для склеивания видеопотоков с использованием данных с нескольких камер. Программа должна обеспечивать выравнивание и наложение кадров видеопотоков с целью создания единого потока видео, который объединяет данные с различных камер в режиме реального времени.

## Этапы обработки видеопотоков

Программа обработки видеопотоков включает в себя следующие этапы:

1. **Выравнивание кадров из разных видеопотоков**: Этот этап включает обнаружение ключевых признаков на кадрах видеопотоков и их сопоставление между разными видеопотоками. Это необходимо для выравнивания кадров относительно друг друга.

2. **Наложение кадров из разных видеопотоков**: На этом этапе происходит наложение выровненных кадров из разных видеопотоков для создания единого потока видео, который объединяет данные с различных камер в единый панорамный видеопоток.
